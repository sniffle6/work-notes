## Highlights

- Added a Done button to inbox cards for completed work.
- Added a dedicated Done view where completed notes remain accessible.

## Changes

- Completed notes are preserved separately from archived notes.
- Added Reopen to return completed notes to the active Inbox.
- Done and Archive now represent distinct outcomes: completed work versus intentionally discarded work.
- Completed-note state persists across application restarts.

## Install And Update Notes

- Windows installer is attached to the GitHub release.
- Existing installs update from the latest-release `latest.json` after publish.
