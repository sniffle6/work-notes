## Highlights
- Launch at startup now works in the installed Windows app.
- Minimize to tray on close now controls whether closing hides the app or exits it.

## Changes
- Added native Windows startup registration for the Launch at startup setting.
- Applied startup and hotkey changes during settings save, with rollback if native registration fails.
- Re-read the minimize-to-tray setting when the main window closes so changes apply without restarting.
- Aligned browser fallback settings with installed-app defaults.

## Install And Update Notes
- Windows installer is attached to the GitHub release.
- Existing installs update from the latest-release `latest.json` after publish.
