## Highlights

- Today is now a full month Calendar that opens on the current day.
- Open and completed tasks appear directly on the dates where they are due or completed.

## Changes

- Added daily Open, Done, Unscheduled, and overdue task buckets.
- Added persisted completion timestamps so finished work appears on the correct calendar date.
- Added task accept, complete, and reopen controls to the Calendar.
- Added clearer month navigation and renamed the Today destination to Calendar.
- Existing completed tasks without a recorded completion date remain available under `Done - date unavailable`.

## Install And Update Notes

- Windows installer is attached to the GitHub release.
- Existing installs update from the latest-release `latest.json` after publish.
