## Highlights

- Calendar tasks now appear on their captured date, with linked due and completion occurrences.
- Selected-day tasks include a visual lifecycle connecting Captured, Due, and Completed dates.

## Changes

- Existing action items are anchored to the date their source note was captured.
- Added linked calendar echoes for due and completion dates, including merged same-day moments.
- Added hover and keyboard-focus highlighting between related calendar occurrences.
- Improved responsive sizing so all six calendar weeks remain visible.
- Simplified the Calendar into a flatter, less card-based design.

## Install And Update Notes

- Windows installer is attached to the GitHub release.
- Existing installs update from the latest-release `latest.json` after publish.
