## Highlights

- Add timestamped notes to existing Work Notes for ongoing context, updates, and decisions.
- Add an optional resolution when marking a note Done.

## Changes

- Added a Notes section to each Work Note with append-only timestamped entries.
- Done now opens a confirmation where a resolution can be recorded.
- Resolution text appears with completed notes and remains available if the note is reopened.
- Local note parsing now uses a consistent pinned Codex model instead of inheriting global defaults.

## Install And Update Notes

- Windows installer is attached to the GitHub release.
- Existing installs update from the latest-release `latest.json` after publish.
