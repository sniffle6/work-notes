## Highlights
- Parser work now shows visible progress while Codex is cleaning notes.
- Notes left mid-parse after restart are recovered and retried instead of staying stuck.
- Timeout failures stop repeating the same long parse attempt and surface as failures the user can retry.

## Changes
- Added parser activity indicators in the sidebar, inbox rows, and note detail view.
- Added completion and failure notifications after parser work finishes.
- Refreshed active parser state while work is in progress so the UI updates without navigation.

- Windows installer is attached to the GitHub release.
