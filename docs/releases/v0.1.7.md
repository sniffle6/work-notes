## Highlights
- Raw note text can now be edited from the note detail view before reparsing.
- Work Notes now uses the refreshed app icon across Windows app assets.

## Changes
- Saving edited raw text queues a fresh parse and clears the previous parser-derived summary, cleaned note text, tags, and unaccepted suggested actions so stale parser output does not look current.
- Accepted action items are preserved when raw text is edited.
- The app icon artwork has less outside padding, making it easier to recognize in the tray, taskbar, and installer surfaces.

## Install And Update Notes
- Windows installer is attached to the GitHub release.
- Existing installs update from the latest-release `latest.json` after publish.
